# Guest Review Sentiment Classifier
## Project Overview
Guest Review Sentiment Classifier is an AI-powered web application designed for Trishul Eco-Homestays. The system helps staff quickly analyze guest reviews by automatically classifying sentiment, identifying review themes, and generating suggested management responses.
## Problem Statement
Trishul Eco-Homestays receives a large number of guest reviews across different platforms. Manually reading and categorizing reviews is time-consuming and inefficient.
## Solution
The application allows staff members to paste one or multiple reviews and receive:
- Sentiment Classification (Positive, Neutral, Negative)
- Theme Detection (Food, Host, Location, Cleanliness, Value, Experience)
- AI-Generated Management Response
## Features
- Single and Batch Review Processing
- Sentiment Analysis
- Theme Classification
- AI Response Generation
- Results Dashboard
